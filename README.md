## What's Different from Claude Code Version?

These skills are **optimized for OpenClaw**:

- ✅ OpenClaw-specific metadata (emoji, requirements)
- ✅ Leaner prose — less context, same frameworks
- ✅ Trimmed promotional content
- ✅ Faster skill loading

For the Claude Code version: [ai-marketing-claude-code-skills](https://github.com/BrianRWagner/ai-marketing-claude-code-skills)

## Quick Install

```bash
# Clone the repo
git clone https://github.com/BrianRWagner/ai-marketing-openclaw-skills.git

# Copy skills to OpenClaw's skills folder
cp -r ai-marketing-openclaw-skills/* ~/.openclaw/skills/
```

OpenClaw will automatically load these skills based on trigger phrases.
